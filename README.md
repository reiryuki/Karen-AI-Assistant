# Karen AI Assistant

## Descriptions
An app powered by Tasker and an importable Tasker project xml

## Features
- Device Assistant
- 2 AI Agents
- Voice & text promt
- Continue (Generate text with history without additional promt.)
- Screen Info
- Screenshot
- Front & rear camera
- Any files
- Previous files
- Modify AI Agents data
- Launch apps & URL
- Multi language
- Text to speech
- Real time, date, news, & weather
- Java (Promt to control device via java code automatically. Does not use root nor adb shell nor Shizuku nor accessibility service permission. Does not allow deleting files nor folders.)
