# Karen AI Assistant

## Descriptions
An app powered by Tasker and an importable Tasker project xml

## Features
- Device Assistant
- Artificial Intelligence
- Voice match
- Keyboard text 
- Screen Info
- Screenshot
- Front & rear camera
- Any files
- Modify data
- Launch apps & URL
- Multi language
- Text to speech
- Real time news & weather
- Java (Promt to control device via java code automatically. Does not use root nor adb shell nor Shizuku nor accessibility service permission. Does not allow deleting files nor folders.)
